## 🚀 Featured Projects

> *Full project list below — these are the ones that show how I think.*

---

### 🛒 Self-Healing E-Commerce Platform
**The platform that fixes itself before you even notice it's broken.**

A microservices e-commerce app (cart, payment, inventory) deployed on EKS with full chaos engineering — pods are intentionally killed and Kubernetes self-healing is demonstrated live on a Grafana dashboard.

`AWS` `EKS` `Docker` `Helm` `Terraform` `GitHub Actions` `Ansible` `Prometheus` `Grafana`

---

### 🤖 GitOps Zero-Touch Deployment Pipeline
**Push code. Everything else happens automatically.**

A fully automated GitOps pipeline where pushing to GitHub triggers Terraform infrastructure changes and ArgoCD syncs Kubernetes manifests automatically. Zero manual intervention from code to production. Slack notifications on every deployment.

`GitHub Actions` `Terraform` `ArgoCD` `Kubernetes` `AWS` `Slack`

---

### 🏗️ Production-Grade Terraform Module Library
**Reusable infrastructure building blocks for dev, staging, and production.**

A library of production-grade Terraform modules (VPC, EKS, RDS, S3, IAM) used across three isolated environments. Remote state in S3 with DynamoDB locking. GitHub Actions posts `terraform plan` results as PR comments automatically.

`Terraform` `AWS` `GitHub Actions` `S3` `DynamoDB`

---

### 🐍 Kubernetes Cost Optimizer
**Because nobody should be paying for idle pods.**

A Python tool that queries the Kubernetes API, identifies over-provisioned pods using less than 20% of their requested CPU/memory, generates optimization recommendations and delivers a weekly report to Slack. Deployed as a CronJob inside the cluster.

`Python` `Kubernetes` `AWS` `Slack API`

---

### ☁️ Automated Disaster Recovery System
**RTO under 5 minutes. Documented and proven.**

Primary infrastructure in us-east-1 with full Terraform replication to eu-west-1. A Python Lambda monitors primary region health and triggers automatic failover if it goes down. Cross-region RDS snapshots included. Recovery Time Objective under 5 minutes.

`AWS Lambda` `Python` `Terraform` `RDS` `Multi-Region` `AWS Route53`
